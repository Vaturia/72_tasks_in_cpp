# Chapter №6
### Exercise №1
Task:
Write a program that compares two numbers entered from the keyboard.
The program should indicate which number is smaller, or, if the numbers are equal, display an appropriate message.
#### Example of program execution result:
```
Enter 2 integer: 48 54
48 less 54
```
### Exercise №2
Task:
Write a program that displays a message on the screen in a "telegraph" style: The letters of the message should appear one at a time with some delay.
#### Example of program execution result:
```
Enter message: Telegraph is cool
Telegraph is cool
```
### Exercise №3
Task:
Write a program that contains current information about ten requests for airline tickets. Each request should have:
* destination;
* flight number;
* full name of the passenger;
* desired departure date.
The program must provide:
* storing all applications in the form of a list;
* adding and deleting applications;
* output of all applications.
#### Example of program execution result:
```
=============================================
=                                           =
=       Air Ticket Managment System         =
=                                           =
=============================================
### Select Action ###
Add application(Enter A/a).
Remove application(Enter R/r).
Print all application(Enter P/p).
Exit(Enter E/e)
Enter your choice: 
```
