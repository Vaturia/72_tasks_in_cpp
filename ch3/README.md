`source file ex1.cpp`
Task:
Write a program that calculates the speed at which a runner covered a distance.
#### Example of program execution result:
`Enter distance (meters): 1000 `
`Enter time (minutes.seconds): 3.25`
`You ran at speed 17.56 km/hour`
### Exercise №2
`source file ex2.cpp`
Task:
Write a program that calculates the sum of the first n positive even integers.
The user enters the number of numbers to be summed.
#### Example of program execution result:
`Enter the number of numbers to be summed: 12`
`Summary of 12 positive even numbers is: 42`
